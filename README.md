# Northstar Operations Performance Dashboard

## Overview
This project demonstrates the development of a full operations performance dashboard using Power BI and SQL.

## Key Features
- Star schema data modeling (fact and dimension tables)
- DAX measures for Revenue, Margin, Margin %, and Variance
- Executive KPI layout with slicers and conditional formatting
- Profitability and performance analysis

## Tools Used
- Power BI
- SQL
- DAX
- Dimensional Modeling
